> NoSQL = Not Only SQL

NoSQL是一类非 SQL（关系）数据库，使用预定义表和列之外的其他数据模型。此类包含许多不同类型的数据库以及数据存储和操作的方法

Nosql四种类型

* 文档数据库：数据存储在 JSON 文档
    * MongoDB
    * DynamoDB

* 键值存储：数据存储在简单的键值对中
    * Redis

* 图数据库：数据表示为由节点和关系构建的网络
    * Neo4J

* 宽列存储：将数据存储在表中
